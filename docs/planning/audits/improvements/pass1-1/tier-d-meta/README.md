## Why These Are "Meta"

Tiers A–C work within UL's framework (questioning axioms, operations, or writing conventions). Tier D questions the framework itself:

- D1 asks: does UL's mathematical structure match its claimed role?
- D2 asks: can UL be wrong? (If not, it's not science)
- D3 asks: what are UL's boundaries? (Where does the universal-and-minimal core end and the domain-specific extension begin?)
