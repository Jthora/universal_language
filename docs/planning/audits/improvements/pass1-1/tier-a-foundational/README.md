## Why These Questions Are Foundational

These are not questions about whether UL is *implemented correctly* (Tier B) or *specified completely* (Tier C). They ask whether UL's starting points are **logically necessary, merely sufficient, or potentially wrong.**

If A1's axiom is unprovable, UL is a *model* (one of potentially many), not *the* theory.  
If A2's geometry is not uniquely forced, UL has a *parameter* (which geometry?) that should be acknowledged.  
If A3's sorts don't survive logic changes, UL is universal only for classical reasoning, not natural language.

## Relationship to Each Other

```
A1 (Is meaning = relationship?)
  ├── If YES (provable): UL's starting point is secure
  ├── If NO (postulate): UL is one model among many
  │     └── Then A2 becomes even more critical (choosing Euclidean = another parameter)
  └── Need counterexample analysis: qualia, indexicals, performatives

A2 (Why Euclidean?)
  ├── If uniquely forced: UL's geometry is determined
  ├── If not forced: UL has a geometry parameter
  │     └── Then we need to characterize what changes for each geometry
  └── 5th postulate (unique parallelism) is the critical test

A3 (Do 4 sorts survive?)
  ├── If YES under all logics: sorts are genuinely universal
  ├── If NO under some logics: sorts are logic-dependent
  │     └── Need to specify which logics UL covers
  └── Classical, intuitionistic, paraconsistent, fuzzy/probabilistic
```
