## The Gap

The current meaning-space is a flat set-theoretic structure. There is no notion of:
- **Distance between meanings** (how far is "happy" from "sad"? from "purple"?)
- **Curvature** (are some regions of meaning-space "tighter" than others?)
- **Geodesics** (what is the shortest semantic path between two concepts?)
- **Parallel transport** (how does a meaning change when moved to a different context?)

Natural language has these properties implicitly. Semantic similarity judgments are distance-like. Metaphor maps meanings along paths. Context shifts meanings continuously. These suggest meaning-space has GEOMETRY, not just topology.

## Existing Work

Expedition-two `metric-and-grounding.md` proves:

- **Theorem 8 (Semantic Distance):** Define $d(x, y)$ as the infimum path length of Σ_UL transformations between $x$ and $y$. This is a pseudometric on $G_e$.
- **Theorem 9 (Metric Completeness):** Under the semantic distance, the completion of $G_e$ is a Polish space.
- **Theorem 10 (Geodesic Existence):** Between any two elements of the completion, there exists a length-minimizing path.
- **Theorem 11 (Curvature Bounds):** Under certain conditions, the semantic space has bounded curvature (bounded above by some $\kappa$).

Additionally, `gauge-bundle-of-meaning.md` introduces:
- A fiber bundle structure where context is the base space and meaning is the fiber
- A connection (parallel transport) that describes how meaning shifts under context change
- Holonomy: the meaning change accumulated by traversing a closed loop in context-space

## Research Questions

### Q1: Is the metric on meaning-space Riemannian?

A Riemannian metric requires a smooth manifold structure and a positive-definite inner product on each tangent space.

**Smoothness problem:** $G_e$ is constructed as a set of geometric configurations. Is this set a manifold? It may have singularities (where configurations collapse) or stratifications (different-dimensioned components). If so, it's not a smooth manifold and Riemannian geometry doesn't apply directly.

**Alternative:** An Alexandrov space (metric space with curvature bounds but without smooth structure). Theorem 11's curvature bounds suggest this may be the right framework.

### Q2: Does the Fisher metric (from 3.1) agree with the intrinsic metric?

Theorem 8 defines a metric via transformation path length. Theorem 3 (from 3.1) defines a metric via Fisher information. Are these the same?

- If yes: powerful result — statistical and geometric notions of semantic distance coincide
- If no: two competing notions of semantic distance, which one captures human intuition better?

### Q3: What is the curvature of meaning-space?

Theorem 11 gives curvature BOUNDS. Can we compute the actual curvature?

**Physical interpretation:** 
- Positive curvature = meanings are "closer together than expected" (convergent region)
- Negative curvature = meanings are "farther apart than expected" (divergent region)  
- Zero curvature = flat, Euclidean-like region

**Prediction:** Abstract conceptual regions (mathematics, philosophy) should have higher curvature (many inter-related concepts in small space), while concrete sensory regions (colors, shapes) should have lower curvature.

### Q4: Does parallel transport provide a theory of context-dependence?

If meaning-space is a fiber bundle (expedition-one):
- **Base space:** Context (situation, discourse state, shared knowledge)
- **Fiber:** Meaning at that context
- **Connection:** How meaning shifts when context shifts

Then parallel transport answers: "If the meaning of 'bank' in financial context is X, what is its meaning when transported to the river context?"

The holonomy (accumulated change around a closed context-loop) would predict: polysemy arises from non-trivial holonomy. This is the Polysemy-Holonomy Theorem (claimed proven in metaphor-formalization.md).

### Q5: Can we recover computational NLP notions?

Modern NLP uses vector embeddings (word2vec, etc.) where:
- Semantic similarity ≈ cosine distance
- Analogies ≈ vector arithmetic (king - man + woman ≈ queen)

If meaning-space is Riemannian:
- Vector embeddings are local linearizations (tangent space at a point)
- The embedding quality depends on how curved the space is locally
- Analogies are parallel transport in the tangent bundle

This would provide a theoretical grounding for why word embeddings work (and when they break down).

## Approach

### Phase A: Manifold structure

1. Characterize the topology of $G_e$ (it's at minimum a metric space from Theorem 8)
2. Determine if it's a manifold, an orbifold, a stratified space, or an Alexandrov space
3. If not a manifold: determine the appropriate generalization of Riemannian geometry

### Phase B: Metric properties

1. Verify Theorems 8-11 against the rigorous semantic domain (Task 2.1)
2. Compute curvature explicitly for small examples (e.g., the meaning-space of a 3-entity, 2-relation model)
3. Compare Fisher metric and intrinsic metric

### Phase C: Fiber bundle formalization

1. Rigorously define the base space (context), fiber (meaning), and total space
2. Define the connection (how meaning shifts with context)
3. Compute holonomy for specific examples (polysemy, metaphor, irony)
4. Verify or refute the Polysemy-Holonomy Theorem

### Phase D: Computational connection

1. State the relationship between Riemannian meaning-space and vector embeddings
2. Derive conditions under which embeddings are faithful (low curvature) vs. distorted (high curvature)
3. Predict where NLP embeddings should fail based on curvature analysis

## Deliverable

1. Classification of meaning-space topology (manifold / orbifold / Alexandrov / other)
2. Verified curvature results with explicit examples
3. Rigorous fiber bundle construction with computed holonomy
4. Theoretical connection to NLP embeddings
