## The Gap

Natural language expresses modality pervasively:
- **Alethic:** "It is necessary / possible that..."
- **Epistemic:** "I know / believe / suspect that..."
- **Deontic:** "You must / should / may..."
- **Temporal:** "It was / will be / has always been the case that..."
- **Counterfactual:** "If X had happened, Y would have..."

UL currently has no explicit mechanism for any of these. Assertions are flat — they hold or they don't.

**Framing question:** Can modal operators (necessity, possibility, knowledge, obligation, tense) be expressed as combinations of the 11 core operations applied to geometric context structures? If so, modal logics are simply Σ_UL-algebras with specific context parameterizations — projections of the core, not additions to it. If irreducible, modality lies beyond the minimal core, and we need to characterize what additional primitives are required.

Gap-analysis.md §1.6 identifies this as a major structural gap.

## Standard Framework: Kripke Semantics

Modal logic is standardly modeled by Kripke frames: a set of "possible worlds" $W$ with an accessibility relation $R \subseteq W \times W$.

- $\Box \phi$ (necessarily $\phi$): $\phi$ holds in all $R$-accessible worlds
- $\Diamond \phi$ (possibly $\phi$): $\phi$ holds in some $R$-accessible world

Different properties of $R$ yield different modal logics:
- $R$ reflexive → T (truth)
- $R$ reflexive + transitive → S4 (knowledge)
- $R$ equivalence relation → S5 (logical necessity)
- $R$ serial → D (deontic obligation)

## Research Questions

### Q1: Can possible worlds be encoded geometrically?

**Candidate:** A "world" is an enclosure — a bounded region of meaning-space. The accessibility relation $R$ is a LINE (relation) between enclosures.

$\Box \phi$ in world $w$ = predicate(embed($\phi$), holds_in, every_accessible_world_from(w))

**Problem:** This requires quantification over worlds — i.e., quantify + embed + predicate in a specific combination. Is this expressible in Σ_UL, or does it require new operations?

### Q2: Are the modal operators new primitives or derivable?

**Option A: Derivable.** Modal operators are specific combinations of existing operations:
$$\Box\phi = \text{quantify}(\forall, \text{modify\_entity}(\text{accessible}, \text{embed}(\phi)))$$

This treats "necessarily $\phi$" as "in all accessible contexts, $\phi$."

**Option B: New primitives.** Add $\Box : a \to a$ and $\Diamond : a \to a$ to Σ_UL, with axioms.

Option A is preferred (doesn't change the signature) but may not capture the full range of modal reasoning.

### Q3: How do different modalities relate to geometric structure?

| Modality | Geometric Interpretation | Kripke Relation |
|----------|------------------------|-----------------|
| Alethic (necessity) | True in every region of meaning-space | $R$ = equivalence |
| Epistemic (knowledge) | True in every epistemically accessible region | $R$ = reflexive + transitive |
| Deontic (obligation) | True in every normatively ideal region | $R$ = serial |
| Temporal (future) | True in every temporally successor region | $R$ = forward-linear |
| Counterfactual | True in the closest region where antecedent holds | $R$ + similarity metric |

**The last one (counterfactual) requires a METRIC on possible worlds.** This connects to Task 3.3 (Riemannian context space). Lewis's closest-world semantics for counterfactuals requires a notion of distance between worlds.

### Q4: Interaction with other operations

Modal operators must interact with the existing operations:
- $\Box(a_1 \wedge a_2)$ should equal $\Box a_1 \wedge \Box a_2$ (normality axiom K)
- $\Box a \to a$ should hold for alethic/epistemic but NOT for deontic (the world isn't always as it should be)
- $\Diamond a = \neg\Box\neg a$ (interdefinability)

These interaction axioms need to be verified against the geometric realization.

## Approach

### Phase A: Possible worlds as enclosures

1. Define: a "modal context" is an enclosure in meaning-space
2. Define: the accessibility relation R as a specific type of relation (line-pattern) between enclosures
3. Verify: the resulting structure satisfies the Kripke frame axioms for at least S5

### Phase B: Encode modal operators

1. Attempt Option A first (derive from existing operations)
2. If expressiveness is insufficient, propose Option B (new primitives)
3. State the encoding explicitly for each modality type

### Phase C: Interaction axioms

1. Verify axiom K: $\Box(\phi \to \psi) \to (\Box\phi \to \Box\psi)$
2. Verify T: $\Box\phi \to \phi$ (for reflexive R)
3. Verify S4: $\Box\phi \to \Box\Box\phi$ (for transitive R)
4. Verify S5: $\Diamond\phi \to \Box\Diamond\phi$ (for equivalence R)

### Phase D: Temporal extension

1. Define temporal accessibility: linear order on worlds
2. Encode tense operators: PAST, FUTURE, ALWAYS, SOMETIMES
3. Verify temporal logic axioms (Prior's tense logic)

## Deliverable

2. Modal operators (at least alethic □ and ◇) either derived or added
3. Verified interaction axioms for at least one modal system
4. Assessment of which modal logics are naturally expressible vs. forced
