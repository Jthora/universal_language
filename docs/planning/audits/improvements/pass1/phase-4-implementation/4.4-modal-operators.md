## The Problem

If Task 3.2 determines that modal operators can be encoded in (or added to) Σ_UL, the implementation needs to support necessity (□), possibility (◇), and potentially epistemic, deontic, and temporal modalities.

**Do NOT implement until 3.2 is complete.** The design depends on whether modal operators are:

- (b) New primitives (signature change — operations 12+)

## Implementation Map

### If Option (a) — Derivable

Modal operators are macros over existing Σ_UL:

```rust
/// □φ = ∀ accessible contexts, φ holds
/// Encoding: quantify(universal, modify_entity(accessible, embed(φ)))
pub fn necessarily(phi: Assertion, context: &ModalContext) -> Assertion {
    let embedded = embed(phi);
    let accessible = modify_entity(context.accessibility_modifier(), embedded);
    quantify(Modifier::universal(), accessible)
}

/// ◇φ = ¬□¬φ
pub fn possibly(phi: Assertion, context: &ModalContext) -> Assertion {
    negate(necessarily(negate(phi), context))
}
```

### If Option (b) — New Primitives

```rust
// New operations added to Σ_UL
pub fn necessitate(a: Assertion) -> Assertion;
pub fn possibilize(a: Assertion) -> Assertion;
```

## Modal Context Types

Different modalities use different accessibility relations:

```rust
pub enum ModalType {
    /// All logically possible worlds (S5: equivalence relation)
    Alethic,
    /// All epistemically accessible worlds (S4: reflexive + transitive)
    Epistemic,
    /// All deontically ideal worlds (D: serial)
    Deontic,
    /// All temporal successors (linear order)
    TemporalFuture,
    /// All temporal predecessors
    TemporalPast,
}

pub struct ModalContext {
    modal_type: ModalType,
    // Accessibility relation encoded as a Σ_UL relation
    accessibility: Relation,
}
```

## Kripke Frame Construction

If Task 3.2 determines that possible worlds = enclosures:

```rust
pub struct KripkeFrame {
    /// Worlds are enclosures in meaning-space
    worlds: Vec<Entity>,  // Each is an enclosure
    /// Accessibility is a relation between worlds
    accessibility: Vec<(Entity, Relation, Entity)>,
}

impl KripkeFrame {
    /// Check: □φ holds at world w
    pub fn necessarily_at(&self, phi: &Assertion, w: &Entity) -> bool {
        self.accessible_from(w)
            .all(|w_prime| self.evaluate(phi, w_prime))
    }
    
    /// Check: ◇φ holds at world w
    pub fn possibly_at(&self, phi: &Assertion, w: &Entity) -> bool {
        self.accessible_from(w)
            .any(|w_prime| self.evaluate(phi, w_prime))
    }
}
```
