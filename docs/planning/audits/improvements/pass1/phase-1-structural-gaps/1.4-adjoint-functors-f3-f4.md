## The Adjoint Functor Theorem (AFT)

The General AFT (Freyd): A functor $U: \mathcal{C} \to \mathcal{D}$ has a left adjoint if:
1. $\mathcal{C}$ is complete (has all small limits)
2. $U$ preserves all small limits (is continuous) 
3. The **solution set condition** holds: for each $D \in \mathcal{D}$, there exists a set $S_D$ of morphisms $f_i: D \to U(C_i)$ such that every $f: D \to U(C)$ factors through some $f_i$.

The Special AFT (for locally small categories with a generator): weaker, but still requires completeness + a set of generators + an intersecting family condition.

## What Needs Checking

For $U_3: \text{Lang}_{\text{Affine}} \to \text{Lang}_{\text{Projective}}$:

### Hypothesis 1: $\text{Lang}_{\text{Affine}}$ is complete

**Question:** Does the category of affine-level UL languages have all small limits?

**Approach:** 
- Products: Is the product of two affine-level languages an affine-level language? (Does the product of two Σ_UL-algebras with affine structure have affine structure?)
- Equalizers: Same question for equalizers.
- If both work, completeness follows (products + equalizers ⟹ all limits).

**Potential issue:** Σ_UL-algebras are multi-sorted. Products of multi-sorted algebras are well-understood (form products sortwise). The question is whether the Erlangen-level constraints (affine invariance, etc.) are preserved under products. If the constraints are equational (defined by equations), they are automatically preserved. If they involve inequational conditions, they may not be.

### Hypothesis 2: $U_3$ preserves small limits

**Question:** Does forgetting projective structure preserve products and equalizers?

**Approach:** Forgetful functors from equational categories (varieties of algebras) always preserve limits. The question is whether the Erlangen level categories ARE varieties.

### Hypothesis 3: Solution set condition

**This is typically the hardest to verify.** 

**Approach:** For each projective-level language $L$, we need a SET (not a proper class) of factorizations through affine-level languages. If the languages have bounded cardinality (e.g., generated by finitely many atoms), this is automatic. If not, it requires a cardinality argument.

**Possible route:** If the categories are locally presentable (every object is a directed colimit of presentable objects), the solution set condition holds automatically. Are these categories locally presentable? If Σ_UL has finitary operations (it does — all 11 operations have finite arity), and the axioms are finitary, then the categories of algebras are locally finitely presentable. This would give the result for free.

## Fallback: Direct Construction

If the AFT route is blocked, the alternative is to construct $F_3$ directly:

Given a projective-level language $L$, construct the "free affine completion" $F_3(L)$:
1. Start with the carriers of $L$
2. Add all elements needed to make the operations respect affine structure
3. Quotient by the affine axioms
4. Prove the universal property

This is a standard free algebra construction. The work is in verifying the details for the specific Erlangen constraints at the affine/projective boundary.

## Same Analysis for $U_4$

$U_4: \text{Lang}_{\text{Projective}} \to \text{Lang}_{\text{Topological}}$ involves forgetting projective structure. The same three hypotheses need checking. The topological level is the coarsest, so $\text{Lang}_{\text{Topological}}$ may have additional complications (topological properties are not always equational).
