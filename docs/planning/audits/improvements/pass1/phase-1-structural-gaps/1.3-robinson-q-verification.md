## Robinson Q — The 7 Axioms

For reference, Robinson's Q consists of:

| Axiom | Statement | Geometric Translation Needed |
|-------|-----------|------------------------------|
| Q1 | Sx ≠ 0 | Successor (translation by 1 unit) never returns to origin |
| Q2 | Sx = Sy → x = y | Translation is injective |
| Q3 | x ≠ 0 → ∃y(Sy = x) | Every non-origin point is one translation from another point |
| Q4 | x + 0 = x | Adding zero-length translation preserves position |
| Q5 | x + Sy = S(x + y) | Addition via iterated translation is associative with successor |
| Q6 | x × 0 = 0 | Zero-fold composition gives zero-length translation |
| Q7 | x × Sy = (x × y) + x | Multiplication via iterated addition |

## Step 2: Geometric Verification Strategy

If the computations need to be produced:

### For Q5: x + Sy = S(x + y)

- **x** = n-fold translation from origin = point at position n·u (where u is unit vector)
- **Sy** = (y+1)-fold translation = point at position (y+1)·u
- **x + Sy** = compose the x-translation with the Sy-translation = translation by (n + y + 1)·u
- **S(x + y)** = successor of the (n + y)-fold translation = translation by (n + y + 1)·u
- **Result:** Both sides give the same point. ✓

This needs to be stated in Σ_UL terms:
- x + Sy = compose(translate^n, compose(translate^y, translate)) — n + (y+1) unit translations
- S(x + y) = compose(translate, compose(translate^n, translate^y)) — 1 + (n+y) unit translations
- Equality holds because compose is associative on translations (prove this from the set-theoretic definition of compose in formal-operations.md)

### For Q7: x × Sy = (x × y) + x

- **x × y** = y-fold composition of the x-translation = translation by (n·y)·u
- **x × Sy** = (y+1)-fold composition of the x-translation = translation by (n·(y+1))·u = (n·y + n)·u
- **(x × y) + x** = compose(translate^(n·y), translate^n) = translation by (n·y + n)·u
- **Result:** Both sides give the same point. ✓

Again, this needs Σ_UL operation decomposition and a proof that compose distributes correctly over iterated composition.

## Key Lemma Needed

**Lemma (Composition Associativity).** For translations τ_a, τ_b, τ_c in G_r:

$$\text{compose}(\text{compose}(\tau_a, \tau_b), \tau_c) = \text{compose}(\tau_a, \text{compose}(\tau_b, \tau_c))$$

**Proof:** By formal-operations.md, compose(γ₁, γ₂) concatenates paths. Path concatenation is associative (standard result from algebraic topology). ∎

This lemma is the engine that makes Q5 and Q7 work. If it's not already proven, prove it.
