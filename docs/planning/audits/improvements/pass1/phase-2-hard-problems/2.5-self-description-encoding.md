# Problem 2.5 — Self-Description Encoding

**Phase:** 2 (Hard Problems)  
**Location:** New document, likely `foundations/self-description.md`  
**Effort:** 1–3 months (Route B), 4–8 months (Route A)  
**Dependencies:** 1.3 (Robinson Q verification — required for Route B)

---

## The Problem

formal-foundations.md §5.1(3) claims: "G can express its own structure." This is presented as PROVEN. It is not.

The current argument is one sentence:
> "Because enclosures can embed arbitrary constructions, the system can encode descriptions of its own primitives, operations, and axioms."

This is hand-waving. Self-description is a precise metamathematical property. It requires:
1. An explicit encoding of the system's syntax within the system
2. A proof that the encoding is faithful (preserves truth)
3. Typically: a Gödel numbering and a representability theorem

None of these exist in the project.

## Why Self-Description Matters

Self-description is not an academic curiosity. It's the property that enables:
- **Self-reference:** The system can talk about itself (required for certain types of reasoning)
- **Gödel incompleteness:** The system is rich enough to be incomplete (a sign of real mathematical depth)
- **Meta-linguistic operations:** An agent using UL can reason about UL itself (essential for self-improvement)

If G CANNOT encode its own structure, UL is not truly universal — it can express everything except itself.

## Two Routes

### Route B: Via Arithmetic (Recommended)

**Prerequisite:** Robinson Q is verified in G (Task 1.3).

**Argument:**
1. G contains a model of Robinson's Q arithmetic (Task 1.3)
2. Robinson's Q is essentially undecidable (standard result)
3. By Gödel's First Incompleteness Theorem, any consistent extension of Q can express but not decide all truths about its own syntax
4. The Gödel numbering assigns a natural number to each formula of Q
5. Since ℕ is constructible in G (numbers-and-computability.md), the Gödel numbers are elements of G
6. Since the primitive recursive functions are representable in Q (standard result), the syntactic operations (substitution, provability predicate, etc.) are expressible as Σ_UL terms

**What needs to be done:**
- Verify that the Gödel numbering maps correctly to elements of $G_e$ (via the geometric ℕ construction)
- Verify that the primitive recursive functions used in self-reference (Sub, Prov, etc.) are expressible as compositions of the 11 Σ_UL operations applied to geometric naturals
- State the Self-Description Theorem: "For every Σ_UL-expression $e$, there exists a Σ_UL-term $\lceil e \rceil \in G_e$ (the Gödel code of $e$) such that the basic syntactic properties of $e$ (being a well-formed expression, having a given sort, being provably equivalent to another expression) are decidable by Σ_UL-expressions applied to $\lceil e \rceil$."

**Effort:** 1–3 months. The heavy lifting is already done by Gödel and the Robinson Q verification. The work is in connecting the dots.

### Route A: Direct Encoding

**Approach:** Without going through arithmetic, directly encode each component of Σ_UL as a geometric construction.

**Encoding scheme:**
1. **Sorts:** Encode {e, r, m, a} as four specific geometric configurations (e.g., four distinguished points, or four specific enclosures)
2. **Operations:** For each of the 11 operations, construct a geometric "template" that represents the operation's type signature and behavior
3. **Axioms:** Encode each axiom of the theory as a geometric assertion
4. **Faithfulness:** Prove that the encoding preserves satisfaction — if an axiom is true in G, its encoding evaluates to a true assertion in G

**Challenges:**
- How to encode "for all entities" geometrically (requires quantify to be well-behaved, which connects back to 2.3)
- How to encode the meta-operation "this expression has sort e" — this is a statement ABOUT expressions, not an expression itself
- How to avoid the liar paradox (the system must not be able to assert its own inconsistency)

**Effort:** 4–8 months. Much harder than Route B because it requires a novel encoding rather than using standard metamathematics.

## Recommendation

**Route B is strongly preferred.** The work is:
1. Complete Task 1.3 (Robinson Q verification) — prerequisite
2. Write the connecting argument from Gödel encoding to Σ_UL terms — ~1 month
3. Verify the primitive recursive functions are Σ_UL-representable — ~1 month
4. State and prove the Self-Description Theorem — ~1 month

Route A should only be pursued if Robinson Q verification fails (meaning G doesn't contain enough arithmetic for Gödel's technique).

## What Self-Description Does NOT Give

Even with self-description proven:
- **G is still incomplete** (Gödel's First Theorem applies — there exist true Σ_UL statements not provable from the axioms)
- **G cannot prove its own consistency** (Gödel's Second Theorem)
- **This is a FEATURE** — it means G is rich enough to exhibit real mathematical phenomena

## Deliverable

1. Self-Description Theorem: precisely stated
2. Proof: via Route B (Gödel encoding through arithmetic) or Route A (direct)
3. Corollary: Gödel's Incompleteness Theorems apply to any sufficiently strong theory of G
4. Updated formal-foundations.md §5.1(3): change from "proven" to actually proven
