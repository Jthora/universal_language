# Problem 2.3 — Semantic Assignment Uniqueness

**Phase:** 2 (Hard Problems)  
**Location:** New document, likely `foundations/assignment-uniqueness.md`  
**Effort:** 3–6 months  
**Dependencies:** 1.5 (weak terminality informs categorical uniqueness)

## The Problem

The current UL system assigns semantic roles to geometric operations:

| Geometric Operation | Semantic Role |
|-------------------|--------------|
| Reflection | Negation |
| Region overlap | Conjunction (AND) |
| Adjacency | Disjunction (OR) |
| Scaling | Quantification |
| Enclosure | Nominalization (embed) |
| Extraction from enclosure | Adjectivalization (abstract) |
| Path concatenation | Relation composition |
| Path reversal | Relation inversion |
| Applying transformation to configuration | Entity modification |
| Applying transformation to path | Relation modification |
| Combining entity + path + entity in enclosure | Predication |

These assignments are presented as natural and intuitive. But they are not proven to be the ONLY valid assignments. This is the bridge between geometry and meaning — and it's held together by analogy, not proof.

## What "Uniqueness" Would Mean

**Strong uniqueness:** The assignment is the ONLY Σ_UL-homomorphism from the semantic algebra to the geometric algebra (up to isomorphism).

**Moderate uniqueness:** The assignment is the only one satisfying a set of NATURAL AXIOMS (which need to be specified).

**Weak uniqueness:** The assignment is the only one satisfying the algebraic constraints of the sort system (operations with signature e × r × e → a can only be mapped to geometric operations with the same sort signature).

## Current Status

**Weak uniqueness is essentially established.** The sort signatures constrain the mapping heavily. `predicate : e × r × e → a` can only map to a geometric operation that takes two configurations, a path, and produces an assertion-region. There aren't many candidates.

**Strong uniqueness is likely false.** Geometric operations have automorphisms. Reflection through the x-axis and reflection through the y-axis are both involutions — either could serve as "negation." The choice of WHICH reflection is conventional (though all choices are equivalent up to rotation).

**Moderate uniqueness is the real question.** Can we specify natural axioms that pin down the assignment?

## Natural Axioms

### For negation (currently: reflection)
- **Axiom N1 (Involution):** negate(negate(a)) = a
- **Axiom N2 (Contradiction):** conjoin(a, negate(a)) = ⊥ (the empty/absurd assertion)
- **Axiom N3 (Excluded middle):** disjoin(a, negate(a)) = ⊤ (the universal/tautological assertion)

**Geometric candidates satisfying N1:** Reflection through any line, rotation by π, point inversion. All are involutions.

**Which also satisfy N2+N3?** Reflection through a line: the intersection of a region and its reflection can be empty (if the region doesn't straddle the line) — not always ⊥. So N2 may fail for some inputs. This needs investigation.

### For conjunction (currently: overlap)
- **Axiom C1 (Commutativity):** conjoin(a₁, a₂) = conjoin(a₂, a₁)
- **Axiom C2 (Associativity):** conjoin(conjoin(a₁, a₂), a₃) = conjoin(a₁, conjoin(a₂, a₃))
- **Axiom C3 (Idempotence):** conjoin(a, a) = a
- **Axiom C4 (Monotonicity):** If a₁ "entails" a₂, then conjoin(a₁, a₃) entails conjoin(a₂, a₃)

**Geometric overlap:** Is overlap (region intersection) commutative? Yes. Associative? Yes. Idempotent? Yes. So overlap satisfies C1–C3. C4 requires defining "entailment" geometrically.

**Alternative candidates:** Convex hull intersection, Minkowski sum, union. Each satisfies SOME of these axioms but not all. Overlap (set intersection) is the ONLY geometric operation satisfying C1–C4 on regions. This could be provable — it would be a genuine uniqueness result for conjunction.

### For quantification (currently: scaling)
- **Axiom Q1 (Monotonicity):** quantify(∀, e) entails quantify(∃, e)
- **Axiom Q2 (Interaction with conjunction):** quantify(∀, e) AND P(x) ⟹ P(e) for any P (universal instantiation)

**Geometric scaling:** Is scaling monotone? Larger scale → larger region → "more" instances. Roughly yes, but the formal connection between area and quantity needs to be established.

**Alternative candidates:** Counting (number of copies), density (points per unit area), measure (Lebesgue measure of region). Each gives a different quantification semantics.

**This is the weakest assignment.** Scaling → quantification is a cognitive metaphor (BIG = ALL). It may not be uniquely determined by axioms.

## Approach

### Step 1: Formalize the axiom systems

For each of the 11 operations, write down the natural axioms its semantic role should satisfy. Sources:
- Boolean algebra axioms for {negate, conjoin, disjoin}
- Category theory for {compose, invert}
- Montague semantics for {predicate, quantify}
- Lambda calculus for {embed, abstract}
- Modification semantics (Parsons, Kratzer) for {modify_entity, modify_relation}

### Step 2: Enumerate geometric candidates

For each operation, list all geometric operations on ℝ² that have the right sort signature AND satisfy the axioms from Step 1.

### Step 3: Prove uniqueness or document freedom

For each operation:
- If only one geometric candidate satisfies the axioms → **FORCED**
- If a finite set of candidates all satisfying the axioms → **CONSTRAINED** (document the equivalence class)
- If infinitely many candidates → **CONVENTIONAL** (the assignment is a choice)

### Step 4: Characterize the equivalence classes

For CONSTRAINED operations, the candidates should be related by automorphisms of G. Prove: "All assignments satisfying axioms A1, ..., An are conjugate under the Euclidean group E(2)." This would mean the assignment is unique up to symmetry — the strongest achievable result when FORCED is impossible.

## Difficulty Assessment

| Operation | Expected Outcome | Difficulty |
|-----------|-----------------|------------|
| predicate | FORCED (only operation with right sort signature) | Easy |
| compose | FORCED (path concatenation is the only associative binary operation on paths) | Medium |
| invert | CONSTRAINED (any involutory path operation; all conjugate) | Medium |
| negate | CONSTRAINED (any involution on assertion-regions; all conjugate under E(2)) | Medium |
| conjoin | LIKELY FORCED (overlap is the only comm+assoc+idempotent operation on regions) | Hard |
| disjoin | De Morgan from conjoin+negate (if independent, check separately) | Medium |
| embed | FORCED (enclosure is the only e←a operation) | Easy |
| abstract | CONSTRAINED (multiple extraction methods; characterize equivalence) | Medium |
| modify_entity | FORCED (sort signature + group action structure) | Medium |
| modify_relation | FORCED (sort signature + group action structure) | Medium |
| quantify | CONVENTIONAL (scaling is one of several viable interpretations) | Hard |

## Deliverable

A document that, for each of the 11 operations:
1. States the natural axioms
2. Enumerates the geometric candidates satisfying those axioms
3. Classifies the operation as FORCED / CONSTRAINED / CONVENTIONAL
4. For CONSTRAINED: proves all candidates are conjugate under E(2)
5. For CONVENTIONAL: documents the degrees of freedom
