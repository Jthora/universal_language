# Problem 2.2 — Algebraic Independence of the 11 Operations

**Phase:** 2 (Hard Problems)  
**Location:** New document, likely `foundations/operation-independence.md`  
**Effort:** 1–3 months  
**Dependencies:** 2.1 (semantic domain needed as testing ground)

---

## The Problem

Σ_UL claims to have exactly 11 operations, and claims this set is minimal (removing any one loses expressiveness). But no formal independence proof exists. 

Some operations look suspiciously similar:
- Is `invert(r)` the same as `modify_relation(reflection_along_path, r)`?
- Is `quantify(m, e)` decomposable into `abstract(e)` + `predicate(...)`?
- Is `disjoin(a₁, a₂)` definable as `negate(conjoin(negate(a₁), negate(a₂)))` (De Morgan)?

If any operation IS definable from the others, Σ_UL is not minimal. The operation should be removed (reducing the signature) or the redundancy should be explicitly acknowledged.

## The Standard Technique

**From universal algebra:** An operation $\omega$ is *independent* of a set of operations $\Omega' = \Omega \setminus \{\omega\}$ if there exists a Σ_UL-algebra $A$ where:
- All operations in $\Omega'$ are well-defined and satisfy the usual properties
- $\omega^A$ differs from every term function built from $\Omega'$

**Construction:** For each $\omega_i$ ($i = 1, \ldots, 11$), exhibit an algebra $A_i$ where $\omega_i$ is "collapsed" (e.g., maps everything to a constant) but the other 10 are non-trivial.

**Analogy:** This is exactly how Post's lattice classifies Boolean functions: {AND, OR, NOT} is a complete and independent set because each has a clone it belongs to that the others don't.

## The Key Suspects

### Suspect 1: conjoin/disjoin redundancy (De Morgan)

If negate is available, then conjoin and disjoin are interdefinable:
$$\text{disjoin}(a_1, a_2) = \text{negate}(\text{conjoin}(\text{negate}(a_1), \text{negate}(a_2)))$$

**This is a real concern.** In classical Boolean logic, {AND, NOT} suffices — OR is derivable.

**Test:** Is the geometric disjoin operation (adjacency) the same as the De Morgan compound (reflect, overlap, reflect)?

In the geometric realization:
- conjoin = overlap (shared region)
- disjoin = adjacency (disjoint but proximal regions)
- negate = reflection

De Morgan compound: reflect a₁, reflect a₂, overlap the reflections, reflect the result.

**If the geometric De Morgan compound EQUALS geometric adjacency**, then disjoin is redundant. If they differ geometrically, disjoin is independent.

This needs to be computed explicitly.

### Suspect 2: invert as modify_relation

`invert(r)` reverses a directed path. Could this be achieved by `modify_relation(m_reverse, r)` for some specific modifier m_reverse?

**If m_reverse is a valid modifier** (an element of $G_m$), then invert is redundant — it's just modify_relation applied to a specific modifier.

**Test:** Is path reversal an element of $G_m$? $G_m$ is defined as invertible geometric transformations of ℝ² at some Erlangen level. Path reversal is NOT a transformation of ℝ² — it's a transformation of the path space. So m_reverse would need to be an operation on paths, not on points.

If $G_m$ only contains point transformations, invert is independent. If $G_m$ also contains path transformations, invert might be derivable.

**Check the definition of $G_m$ in formal-operations.md carefully.**

### Suspect 3: quantify decomposition

`quantify(m, e) → a` takes a modifier and an entity and produces an assertion. Could this be: `predicate(modify_entity(m, e), some_relation, some_entity)` — i.e., first modify the entity, then make a predication?

**This would require:** A canonical "existence relation" r_exists and a canonical "universe entity" e_universe, so that: quantify(∀, dogs) = predicate(modify_entity(∀, dogs), exists_in, universe).

**The question is whether r_exists and e_universe are canonical elements of G or ad hoc additions.** If they must be added, quantify is doing something the other operations can't.

## Construction Strategy

For each operation $\omega_i$, construct a quotient algebra $A_i = G / {\sim_i}$ where:
- The equivalence relation $\sim_i$ collapses $\omega_i$ to trivial behavior (e.g., $\omega_i^{A_i}$ is constant)
- The other 10 operations remain non-trivial in $A_i$

If such a quotient exists for all 11, they're independent. If constructing the quotient for some $\omega_i$ forces another operation to also collapse, those operations are linked (potentially derivable).

## Deliverable

1. For each of the 11 operations: either a proof of independence (via counterexample algebra) or a derivation from the other 10
2. If any operations are derivable: a proposed minimal subset and a proof that this subset IS independent
3. Updated Σ_UL signature reflecting the minimal independent set

## Possible Outcomes

| Outcome | Consequence |
|---------|------------|
| All 11 independent | Σ_UL is minimal as claimed. Strong result. |
| 10 independent, 1 derivable (e.g., disjoin) | Remove disjoin from Σ_UL, define as De Morgan compound. Minor signature change. |
| 9 independent, 2 derivable | Significant simplification. Rewrite formal foundations with 9 ops. |
| 8 or fewer independent | Major revision needed. The "11 operations" claim was wrong. |

All outcomes are scientifically valuable. Finding the ACTUAL minimal set is more important than defending the current count.

**Note:** If any operations reduce to others, the result STRENGTHENS UL's universality claim — it doesn't weaken it. A smaller minimal set means more phenomena can be expressed in fewer operations. Fewer primitives = greater universality. The initial object becomes simpler, and every Σ_UL-algebra (every particular framework) is still a projection of it.
