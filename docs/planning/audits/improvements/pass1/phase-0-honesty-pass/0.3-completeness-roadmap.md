## Specific Additions

### New subsection: "§3.8 Open Problems in Completeness"

```markdown
### 3.8 Open Problems in Completeness

> **Status: OPEN.** The coverage argument in §3.1–3.6 demonstrates that
> the 11 operations cover all sort-transitions in Σ_UL. This is necessary
> but not sufficient for completeness. The following problems remain open.

#### Open Problem C1: Semantic Domain Axiomatization

**Statement:** Give a formal definition of the class of "meaning structures"
that Σ_UL is claimed to be complete over.

**Discussion:** The claim "the 11 operations suffice to express all finite
relationships between distinguishable entities" is undefined until "all
finite relationships" is axiomatized. The semantic domain might be:

- (a) All finite labeled hypergraphs with nodes typed by {E, R, M, A}
  and edges respecting operation arities
- (b) All relational structures over finite signatures with 4 sorts
- (c) All finite Σ_UL-terms (trivially complete by definition,
  but vacuous)
- (d) Some other class motivated by natural language semantics

Option (a) is the most natural. It makes completeness a concrete question:
can every such hypergraph be constructed from atoms using the 11 operations?

**Impact:** Without this definition, "completeness" is literally undefined.

**Cross-reference:** Phase 2 Problem 2.1

#### Open Problem C2: Algebraic Independence

**Statement:** Prove that no Σ_UL operation is definable as a term
built from the other 10 operations.

**Discussion:** If some operations are derivable, the signature is
redundant. For example:
- Is `quantify(m, e)` equivalent to `predicate(modify_entity(m, e), r, e')`
  for some choice of r and e'?
- Is `invert(r)` equivalent to `modify_relation(reflection, r)`?

If any operation IS derivable, Σ_UL should be reduced to its minimal
independent subset. If all 11 are independent, this should be proven
by exhibiting, for each operation oᵢ, a Σ_UL-algebra where oᵢ behaves
differently from any term in the other 10.

**Impact:** Determines whether Σ_UL is minimal as claimed.

**Cross-reference:** Phase 2 Problem 2.2

#### Open Problem C3: Transitivity of the Operation Monoid

**Statement:** Prove that the monoid generated by the 11 operations
acts transitively on the set of all meaning structures (per the
axiomatization in C1).

**Discussion:** This is the strongest form of completeness: not just
that every structure is reachable from SOME starting point, but that
the operations can transform any structure into any other. This may be
too strong — a weaker but still useful result would be that every
structure is reachable from the set of atomic generators.

**Cross-reference:** Phase 2 Problems 2.1, 2.2
```
