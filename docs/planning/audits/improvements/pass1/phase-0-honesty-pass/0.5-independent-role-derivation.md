## Objective

Attempt to derive the 5 role properties (dependency rank, dimensionality, parameterization, symmetry, boundary) from each of the 5 intellectual traditions analyzed in independent-derivation.md, **without using geometric vocabulary**.

This is the critical test of whether the Unique Grounding Theorem is discovered or imposed.

## The 5 Role Properties to Derive

From formal-foundations.md §4A (geometric):

| Property | Geometric Definition |
|----------|---------------------|
| **Dependency rank** | Number of prior primitives required (Point=0, Line=2 points, Angle=2 lines, ...) |
| **Dimensionality** | Geometric dimension (Point=0, Line=1, Angle=angle-space, Curve=1-param, Enclosure=2) |
| **Parameterization** | Whether the primitive requires a continuous parameter |
| **Symmetry** | Size of the automorphism group |
| **Boundary** | Whether the primitive creates/requires a boundary |

## Derivation Attempt per Tradition

### From Frege (1879, 1892)

**Available vocabulary:** Concept, Object, Function, Argument, Saturation, Sense, Reference

**Attempt:**
- **Dependency rank equivalent:** Frege's concept/object distinction gives a natural arity. Objects are 0-place (saturated). First-order concepts are 1-place functions. Second-order concepts take concepts as arguments. This IS a dependency hierarchy — but it's ARITY, not dependency rank. Can arity be identified with dependency rank?
  - Objects = 0 arguments needed → dependency rank 0 ✓
  - Concepts = 1+ arguments → dependency rank > 0 ✓
  - But: Frege distinguishes first-order vs second-order concepts by what they take as argument, not by how many primitives construct them. Arity ≠ construction dependency.
- **Symmetry equivalent:** Frege's objects have maximal identity conditions (Leibniz's law). Concepts can have "the same extension" but different senses. This gives a coarser notion of identity for concepts → smaller symmetry group? Plausible but indirect.
- **Parameterization:** No clear Fregean analogue. Concepts are not "parameterized" in Frege.
- **Boundary:** No clear Fregean analogue.

**Verdict:** Dependency rank PARTIALLY derivable (via arity). Others: NOT derivable from Frege alone.

### From Montague (1970, 1973)

**Available vocabulary:** Type hierarchy, Denotation, Intensional operator, Lambda abstraction, Extension, Intension

**Attempt:**
- **Dependency rank:** Montague's type system gives explicit construction order: e (entities) and t (truth values) are basic. Functions e→t, (e→t)→t, etc. are built from them. The TYPE ORDER is a dependency rank.
  - Type e = rank 0 ✓
  - Type e→t = rank 1 (needs an entity argument) ✓
  - Type (e→t)→t = rank 2 ✓
  - This works, but only for the first 3 levels.
- **Parameterization:** Montague's intensional operators (^, ˇ) involve worlds as parameters. Intensions are functions from possible worlds → extensions. This IS parameterization — the possible-world index is a continuous (or at least structured) parameter.
  - Intensions = parameterized ✓ (maps to Curve)
- **Boundary:** Montague's sets have boundaries (membership is crisp). Extensions are sets. But this is standard set theory, not a semantic primitive.
- **Symmetry:** Not clearly present in Montague.

**Verdict:** Dependency rank DERIVABLE (via type order). Parameterization DERIVABLE (via intension). Symmetry, boundary: NOT derivable from Montague alone.

### From Jackendoff (1983, 1990)

**Available vocabulary:** Conceptual Structure, Thematic roles (Agent, Theme, Goal, Source, Path), Event structure, Aspectual features

**Attempt:**
- **Dependency rank:** Things participate in Events. Events have Thematic roles. Roles relate Things to Events. Natural order: Thing (rank 0) → Event (needs participants, rank > 0) → Role (connects Thing and Event, rank > both). This loosely fits.
- **Parameterization:** Events have temporal structure (bounded/unbounded, telic/atelic). Unbounded events = ongoing process = requires time parameter. This is PARAMETERIZATION in the linguistic sense.
  - Activities (run, push) = parameterized by time ✓ (maps to Process/Curve)
  - Achievements (arrive, die) = instantaneous = non-parameterized ✓ (maps to Point-like)
- **Boundary:** Telic events have endpoints. Atelic events don't. The telic/atelic distinction IS a boundary property — telicity = having a natural boundary.
  - "Run" (atelic) = no boundary
  - "Run a mile" (telic) = has boundary ✓
- **Symmetry:** Not clearly present.

**Verdict:** Dependency rank PARTIALLY derivable. Parameterization DERIVABLE. Boundary DERIVABLE (via telicity). Symmetry: NOT derivable.

### From Langacker (1987)

**Available vocabulary:** Profile, Base, Trajector, Landmark, Schema, Scanning (sequential/summary), Construal

**Attempt:**
- **Dependency rank:** Langacker's Trajector/Landmark asymmetry gives a primitive ordering: entities are profiled against their base. Simple entities are autonomous. Relations require both a trajector and a landmark. Modifiers construe relations. This IS a dependency hierarchy.
- **Parameterization:** Sequential scanning = tracking a relation through time = parameterized processing. Summary scanning = simultaneous = non-parameterized.
  - Sequential scanning ↔ Curve (parameterized) ✓
  - Summary scanning ↔ Angle (instantaneous comparison) ✓
- **Boundary:** Bounded vs unbounded in Langacker maps directly to Lindner (1982) container schemas. "In", "out", "boundary."
- **Symmetry:** Langacker's figure/ground asymmetry. The trajector IS the figure — it has a distinguished role. This breaks symmetry specifically for relations (they have an inherent direction). Points/entities have maximal symmetry (no distinguished internal structure).

**Verdict:** ALL 5 PROPERTIES at least partially derivable from Langacker. This is the strongest tradition for this purpose.

### From Wierzbicka (1972, 1996)

**Available vocabulary:** Natural Semantic Metalanguage (NSM) primes: SOMEONE, SOMETHING, THIS, DO, HAPPEN, MOVE, THERE IS, etc.

**Attempt:**
- **Dependency rank:** NSM primes are defined as indefinable — they CAN'T be constructed from each other. This gives rank 0 to all primes equally. NSM doesn't have a construction hierarchy.
- **Parameterization:** HAPPEN and MOVE imply temporal extent (parameterization), but this is built into the prime, not a property extracted from it.
- **Boundary:** Not present as a formal property.
- **Symmetry:** Not present.

**Verdict:** Dependency rank NOT derivable (all primes are equally primitive in NSM). Others: NOT derivable from NSM.

## Honest Assessment

- **Dependency rank** is independently motivated from 2–3 traditions (Montague's type hierarchy, Langacker's T/L hierarchy). This is the STRONGEST role property — it's not circular.
- **Parameterization** is independently motivated from 3 traditions (Montague intension, Jackendoff aspect, Langacker scanning). Also strong.
- **Boundary** is independently motivated from 2 traditions (Jackendoff telicity, Langacker boundedness). Moderate support.
- **Symmetry** is weakly supported (only Langacker, and that's figure/ground asymmetry, not automorphism groups). The identification of symmetry with the formal notion of automorphism group IS an imposition from geometry.
- **Dimensionality** is NOT derivable from any tradition without geometry. This is the weakest role property — it may be purely geometric with no independent semantic motivation.

## What This Means

If the Unique Grounding Theorem still holds using only {dependency rank, parameterization, boundary} — the three independently motivated properties — then the grounding is non-circular.

If the theorem REQUIRES dimensionality or symmetry — the geometrically imported properties — then the grounding has a residual circularity that needs to be acknowledged.
