## The Gap

In §3.3 (Geometric Universality Theorem), Step 4 proves injectivity of the embedding φ: E ↪ G. The current argument:

> "Geometric constructions with different components are visually — and thus mathematically — distinct."

This is a conceptual leap. "Visually distinct" is not a defined mathematical property. "Set-theoretically distinct" is. The leap from one to the other is not justified.

## The Fix

The fix already exists in the project — it just isn't cited.

**formal-operations.md §2 (Operation Distinctness Theorem)** proves:

> Each of the 11 Σ_UL operations preserves input distinctness. If inputs are distinct compact subsets of ℝ², outputs are distinct compact subsets of ℝ².

This is exactly the property needed. The argument:

1. φ maps atoms to distinct points/paths/transformations in G (by construction — Step 2 assigns distinct geometric objects to distinct atoms)
2. φ extends to compound expressions by applying Σ_UL operations (Step 3 — universal property)
3. Σ_UL operations preserve distinctness (Operation Distinctness Theorem from formal-operations.md)
4. Therefore, distinct expressions map to distinct geometric objects (injectivity)
