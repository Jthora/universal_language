## The Problem

formal-foundations.md §4 currently presents the Unique Grounding Theorem as a single monolithic argument:

1. Define 5 geometric role properties (§4.1–4.2)
2. Define 5 semantic role properties (§4.3–4.4) 
3. Show the mapping is forced by exhaustive elimination (§4.5)

The issue is that the semantic role properties in §4.3–4.4 are **defined using the same vocabulary** as the geometric role properties: dependency rank, dimensionality, parameterization, symmetry, boundary status.

- "Existence has dependency rank 0" — this is a definition imposed by the project
- "Points have dependency rank 0" — this is a geometric fact
- "Therefore Existence ↔ Point" — this follows trivially because the definitions were designed to match

The proof IS valid — given the definitions, the elimination IS forced. But the definitions are the load-bearing assumptions, and they're not flagged as such.

## Objective

Split §4 into three clearly separated components:

### Component A: Geometric Role Properties (§4.1–4.2 → §4A)

**Status: PROVEN (geometric facts)**

These are properties of the geometric primitives proved from their mathematical definitions:
- Points have dependency rank 0 (no other primitive needed)
- Lines require exactly 2 points
- Angles require exactly 2 lines
- Curves require parameterization
- Enclosures require boundaries (Jordan Curve Theorem)

These are standard geometric results. No controversy.

### Component B: Semantic Role Properties (§4.3–4.4 → §4B)

**Status: DEFINITION (adopted assumptions)**

These are formal characterizations of semantic primitives:
- Existence has dependency rank 0
- Relation requires exactly 2 entities
- Quality is a comparison between relations
- Process requires parameterization
- Concept requires boundary

**Critical note to add:**

> These characterizations are the load-bearing assumptions of the Unique Grounding Theorem. They are not self-evident — they are interpretive choices formalized as definitions. The key question is whether they are DISCOVERED (following necessarily from the nature of meaning) or IMPOSED (designed to match the geometric properties in §4A).
>
> **Evidence for "discovered":** `independent-derivation.md` §2.1–2.6 shows five independent intellectual traditions (Frege, Montague, Jackendoff, Langacker, Wierzbicka) converging on the same 4-sort structure without geometric vocabulary. This convergence is genuine evidence that the 4-sort structure is natural.
>
> **Evidence for "imposed":** The specific role properties (dependency rank, parameterization, etc.) are formulated in geometric language. The question is not WHETHER Existence and Points are similar — the question is whether "dependency rank 0" is the CORRECT formalization of what makes Existence fundamental, or merely a description crafted to match.
>
> **Open question:** Can these role properties be derived from each tradition independently, using only the vocabulary of that tradition? See Phase 0.5 for the attempt.

### Component C: The Elimination Proof (§4.5 → §4C)

**Status: PROVEN — given the definitions in §4B**

The exhaustive elimination over 5×5 primitive pairs. This proof is correct. Given the role-property definitions, the mapping is unique. The proof itself needs no changes — only the status label needs to reflect its dependence on §4B.

**Revised theorem statement:**

> **Unique Grounding Theorem (Conditional Form).** Let the geometric primitives be characterized by the role properties in §4A, and let the semantic primitives be characterized by the role properties in §4B. Then the mapping G1↔S1, ..., G5↔S5 is the unique role-property-preserving bijection between {Point, Line, Angle, Curve, Enclosure} and {Existence, Relation, Quality, Process, Concept}.

This makes the conditional nature explicit while preserving the correct proof.
