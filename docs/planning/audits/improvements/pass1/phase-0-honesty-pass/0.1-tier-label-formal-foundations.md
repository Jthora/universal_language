### §1.5 (Σ_UL Minimality)
- **Current presentation:** "We claim Σ_UL is the universal linguistic signature" — presented as established.
- **Action:** Split into two claims:
  1. **Sort minimality** (removing any sort loses expressiveness) — `**Status: PROVEN**` — the argument that removing entities means you can't refer to things is trivially correct.
  2. **Operation minimality** (removing any operation loses expressiveness) — `**Status: CONDITIONAL**` — the per-operation necessity arguments are plausible but informal. Formal algebraic independence proof needed (see Phase 2.2).
- **Add note:** "Operation minimality is a coverage argument. See `formal-operations.md` §3.7 and Phase 2 Problem 2.2 for the formal proof roadmap."
- **Reframe "universal linguistic signature":** The claim is NOT that Σ_UL is "the one true language." It is that Σ_UL is the minimal algebraic skeleton shared by all compositional meaning systems — the initial object in Lang(Σ_UL). Every particular framework (first-order logic, DRT, Montague semantics, natural languages) is a Σ_UL-algebra: a specialization with additional domain-specific structure. This reframing must be explicit in the edited text.

## Format Convention

Every section header gets a status block immediately below, formatted as:

```markdown
### N.M [Title]

> **Status:** PROVEN / CONDITIONAL / OPEN  
> **Depends on:** [list of prior results or definitions]  
> **Required for:** [list of downstream results]  
> **Convention note:** [if applicable — marks semantic interpretations that are choices, not necessities]
```
