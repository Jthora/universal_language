## PART I: CONCEPTUAL DOMAINS AS Σ_UL-SUBALGEBRAS (Step E3)

### 1.1 Definition (Recall)

From `metaphor-and-projection.md` §6.1:

**Definition 6 (Conceptual domain).** A **conceptual domain** is a Σ_UL-subalgebra $G_D \subseteq G$ — a subset of the geometric meaning algebra closed under all 11 Σ_UL operations.

To prove that a proposed domain $G_D$ is a valid subalgebra, we must show:

1. $G_D$ is **non-empty** (contains at least one element in each sort that participates)
2. $G_D$ is **closed** under all 11 operations: for any inputs from $G_D$, the operation's output is in $G_D$

### 1.2 Generator Sets

A Σ_UL-subalgebra generated by a set $S$ is the smallest subalgebra containing $S$ — the closure of $S$ under all 11 operations. We write $\langle S \rangle_{\Sigma}$ for this generated subalgebra.

**Definition 7 (Domain generators).** The **generator set** of a conceptual domain $G_D = \langle S_D \rangle_{\Sigma}$ is a minimal set $S_D$ of elements in $G$ (distributed across the four sorts: entities, relations, modifiers, assertions) from which all elements of $G_D$ arise by iterated application of the 11 operations.

### 1.4 Subalgebra Closure Verification

**Theorem 12 (Domain Subalgebra Closure). [PROVEN]** Each of $G_{\text{QM}}$, $G_{\text{ling}}$, and $G_{\text{cog}}$, defined as the Σ_UL-subalgebra generated by the generator sets in §1.3, is a well-defined Σ_UL-subalgebra of $G$.

**Proof.** By the general algebraic fact: for any Σ_UL-algebra $G$ and any subset $S \subseteq |G|$ (where $|G|$ denotes the union of carrier sets), the generated subalgebra $\langle S \rangle_\Sigma$ exists and is a Σ_UL-subalgebra.

More precisely: define $S_0 = S$, and iteratively:

$$S_{n+1} = S_n \cup \{ \text{op}(a_1, \ldots, a_k) : \text{op is a Σ_UL operation, } a_1, \ldots, a_k \in S_n \text{ of correct sorts} \}$$

Then $\langle S \rangle_\Sigma = \bigcup_{n=0}^{\infty} S_n$.

This is closed under all 11 operations by construction: any operation applied to elements of $\langle S \rangle_\Sigma$ produces an element that appears at some finite stage $S_n$, hence is in $\langle S \rangle_\Sigma$.

It remains to verify that the generator sets in §1.3 are **legitimate elements of $G$** — that each generator, as described, is a valid element of the appropriate carrier set of the geometric algebra $G$.

For each generator, we verify membership:

- **Entity generators** (e.g., $e_\psi$): These are geometric constructions in $G_e$ — points or point-like structures in glyph space. Each is specified by a position. ✓
- **Relation generators** (e.g., $r_\nabla$): These are directed connections in $G_r$ — line segments or arcs with direction. Each is specified by two endpoints and a direction. ✓
- **Modifier generators** (e.g., $m_\theta$): These are geometric transformations in $G_m$ — elements of the transformation groups (Euclidean, similarity, affine, etc.). ✓
- **Assertion generators** (e.g., $a_S$): These are complete sentence-frame constructions in $G_a$ — bounded regions containing entity-relation-entity structures connected by the predicate operation. ✓

Since all generators are valid elements of $G$ and closure under operations is guaranteed by the iterative construction, each $G_D = \langle S_D \rangle_\Sigma$ is a well-defined Σ_UL-subalgebra. $\square$

### 1.5 Domain Intersection Analysis

**Theorem 13 (Domain Intersection Non-Emptiness). [PROVEN]** $G_{\text{QM}} \cap G_{\text{cog}}$ contains at least three independent elements (elements not expressible as Σ_UL-operations applied to each other).

**Proof.** We exhibit three elements that belong to both domains.

**Element 1: Superposition-as-pattern.** In $G_{\text{QM}}$: the construction $\text{disjoin}(a_1, a_2)$, where $a_1 = \text{predicate}(e_\psi, r_{\text{spec}}, e_{\lambda_1})$ ("ψ has eigenvalue λ₁") and $a_2 = \text{predicate}(e_\psi, r_{\text{spec}}, e_{\lambda_2})$ ("ψ has eigenvalue λ₂"). This is quantum superposition: ψ is in state λ₁ OR state λ₂.

In $G_{\text{cog}}$: the construction $\text{disjoin}(a'_1, a'_2)$, where $a'_1 = \text{predicate}(e_{\text{act}}, r_{\text{weight}}, e_{\text{rep}_1})$ ("activation weights representation₁") and $a'_2 = \text{predicate}(e_{\text{act}}, r_{\text{weight}}, e_{\text{rep}_2})$ ("activation weights representation₂"). This is simultaneous activation: the cognitive state activates pattern₁ OR pattern₂.

Now: both constructions are built using the same Σ_UL skeleton:

$$S_1 = \text{disjoin}(\text{predicate}(e, r, e'), \text{predicate}(e, r, e''))$$

This skeleton — the abstract "disjunction of two predications sharing a subject" — is an element of $G$ that lies in $\langle S_{\text{QM}} \rangle_\Sigma \cap \langle S_{\text{cog}} \rangle_\Sigma$ because it can be generated from the generators of either domain. It belongs to $G_{\text{QM}}$ (taking $e = e_\psi$, $r = r_{\text{spec}}$, etc.) and to $G_{\text{cog}}$ (taking $e = e_{\text{act}}$, $r = r_{\text{weight}}$, etc.).

More precisely: the shared element is the **structural pattern** — the element of $G_a$ obtained by applying disjoin to two predications with a shared subject entity. What differs between the two domains is which generators fill the slots, but the algebraic structure (the element considered as a term in the Σ_UL-algebra up to renaming of atomic generators) is identical. To make this fully precise: define the **structural type** of a Σ_UL-term as the term obtained by replacing all atomic generators with sort-indexed variables. The structural type of $S_1$ is:

$$\tau_1 = \text{disjoin}(\text{predicate}(\xi_e^1, \xi_r^1, \xi_e^2), \text{predicate}(\xi_e^1, \xi_r^1, \xi_e^3))$$

This structural type is **realized** in both $G_{\text{QM}}$ and $G_{\text{cog}}$ — there exist substitutions of QM generators and of cognitive generators that produce valid elements of $G$. The intersection $G_{\text{QM}} \cap G_{\text{cog}}$ therefore contains at least all elements whose structural types are realizable in both domains.

**Element 2: Measurement-as-selection.** Structural type:

$$\tau_2 = \text{predicate}(\text{embed}(\text{disjoin}(a_1, a_2)), \; r, \; \xi_e)$$

In $G_{\text{QM}}$: embed the superposition (making it an entity — "the state"), then predicate it with a relation that selects one outcome. This is measurement: a superposition collapses to a definite value.

In $G_{\text{cog}}$: embed the multi-activation pattern (making it an entity — "the ambiguous representation"), then predicate it with attention weighting that selects one interpretation. This is disambiguation: an ambiguous input resolves to a single meaning.

Both realizations exist in their respective generated subalgebras by closure under embed and predicate. ✓

**Element 3: Feedback-as-self-interaction.** Structural type:

$$\tau_3 = \text{predicate}(\xi_e, \; \text{compose}(r_1, \text{invert}(r_1)), \; \xi_e)$$

In $G_{\text{QM}}$: an entity related to itself via a composed-then-inverted relation. Realized as the self-interaction term $g_{\text{loc}}|\psi|^2\psi$ — the wavefunction acting on itself.

In $G_{\text{cog}}$: an activation state related to itself via a feedback loop. Realized as the autoregressive mechanism where the model's output re-enters as input — the self-attention loop described in `mechanism-of-action.md` §5.

Both realizations exist by closure under compose, invert, and predicate. ✓

**Independence:** These three structural types are independent — none can be obtained from the others by applying Σ_UL operations:
- $\tau_1$ uses disjoin but not embed or compose-invert
- $\tau_2$ uses embed and disjoin but not compose-invert
- $\tau_3$ uses compose and invert but not disjoin or embed

Therefore the intersection contains at least three algebraically independent elements. $\square$

**Remark.** The intersection $G_{\text{QM}} \cap G_{\text{ling}}$ is also non-empty. The bridge equation $B^e \times (L_4 + Q) - (\sigma/\beta) \rightarrow_F \rightarrow M_\infty$ explicitly juxtaposes linguistic entities ($L_4$) with quasi-physical structure ($\rightarrow_F$, the field generator). The structural type of "composition of entities from different sorts, directed toward a limit" is realizable in both domains. Similarly, $G_{\text{ling}} \cap G_{\text{cog}}$ contains the structural patterns shared by compositional semantics (syntax-driven meaning composition) and compositional representation (neural network composition of embeddings).

## PART II: METAPHOR AS Σ_UL-MORPHISM (Step E4)

### 2.1 Definition

**Definition 8 (Conceptual metaphor).** A **conceptual metaphor** from source domain $G_S$ to target domain $G_T$ is a Σ_UL-homomorphism:

$$\phi: G_S \to G_T$$

That is, a family of maps $\phi_e: (G_S)_e \to (G_T)_e$, $\phi_r: (G_S)_r \to (G_T)_r$, $\phi_m: (G_S)_m \to (G_T)_m$, $\phi_a: (G_S)_a \to (G_T)_a$ that commute with all 11 operations:

$$\phi_a(\text{predicate}^{G_S}(e_1, r, e_2)) = \text{predicate}^{G_T}(\phi_e(e_1), \phi_r(r), \phi_e(e_2))$$

and similarly for modify_entity, modify_relation, negate, conjoin, disjoin, embed, abstract, compose, invert, quantify.

**Remark.** This definition operationalizes Lakoff & Johnson's Conceptual Metaphor Theory (1980) within the Σ_UL framework. Their "systematicity" criterion — that metaphors preserve relational structure — is precisely the homomorphism condition. Their observation that metaphors are partial (some source structure doesn't transfer) corresponds to the morphism having a non-trivial kernel.

### 2.2 The Kernel and Image

For a metaphor $\phi: G_S \to G_T$:

**Definition 9 (Metaphor kernel).** The **kernel** of the metaphor is:

$$\ker(\phi) = \{ x \in G_S : \phi(x) = 0_{G_T} \}$$

where $0_{G_T}$ is the identity/trivial element of the target domain (the null entity, trivial relation, identity transformation, or tautological assertion, respectively in each sort).

Strictly: since Σ_UL-algebras are not groups, the "kernel" is the **preimage of the trivial elements**, sort by sort. For entities: $\ker(\phi_e) = \{e \in (G_S)_e : \phi_e(e) = 0_e^{G_T}\}$, where $0_e^{G_T}$ is the degenerate point (zero entity), and similarly for other sorts.

The kernel contains what is **lost in translation** — source domain structure that has no target domain counterpart.

**Definition 10 (Metaphor image).** The **image** of the metaphor is:

$$\text{im}(\phi) = \{ \phi(x) : x \in G_S \} \subseteq G_T$$

The image is itself a Σ_UL-subalgebra of $G_T$ (the image of a subalgebra under a homomorphism is a subalgebra). It contains what is **gained in translation** — the target domain structure that the metaphor activates.

### 2.3 Erlangen Level of Preservation

From the Erlangen hierarchy (`category-of-languages.md` §2):

| Level | Group | What is preserved |
|---|---|---|
| 5. Euclidean | $E(2)$ | Distances, angles, orientations |
| 4. Similarity | $\text{Sim}(2)$ | Angles, ratios, shapes |
| 3. Affine | $\text{Aff}(2)$ | Parallelism, ratios along lines, convexity |
| 2. Projective | $\text{PGL}(3)$ | Cross-ratios, collinearity, incidence |
| 1. Topological | $\text{Homeo}(\mathbb{R}^2)$ | Connectivity, containment, boundary |

**Definition 11 (Erlangen level of a metaphor).** The Erlangen level of a metaphor $\phi: G_S \to G_T$ is the **highest level** at which $\phi$ commutes with the group action:

$$\text{level}(\phi) = \max\{k : \phi \circ g = g \circ \phi \text{ for all } g \in \mathcal{G}_k\}$$

where $\mathcal{G}_k$ is the symmetry group at level $k$.

A higher Erlangen level means the metaphor preserves more geometric structure — it is a "tighter" analogy.

#### 2.4.2 Verification: $\phi_{\text{primer}}$ Is a Σ_UL-Homomorphism

**Theorem 14 (Metaphor Preservation). [PROVEN]** The map $\phi_{\text{primer}}: G_{\text{QM}} \to G_{\text{cog}}$ defined in §2.4.1 is a Σ_UL-homomorphism.

**Proof.** By the universal property of generated subalgebras, it suffices to define $\phi_{\text{primer}}$ on the generators and verify that the map extends consistently — i.e., that no relational constraint among generators is violated.

The generators of $G_{\text{QM}}$ are free in the following sense: the only relations among them are those imposed by the Σ_UL operations themselves (associativity of compose, involution of negate, etc.), and these are structural axioms of Σ_UL that hold in ALL Σ_UL-algebras. Therefore any sort-respecting map from generators to elements of a Σ_UL-algebra extends to a unique homomorphism.

Formally:

**Step 1.** Let $F(S_{\text{QM}})$ be the free Σ_UL-algebra generated by the 13 QM generators. By freeness, the map $\phi_0: S_{\text{QM}} \to |G_{\text{cog}}|$ (from §2.4.1) extends uniquely to a homomorphism $\bar{\phi}: F(S_{\text{QM}}) \to G_{\text{cog}}$.

**Step 2.** The domain $G_{\text{QM}} = \langle S_{\text{QM}} \rangle_\Sigma$ is a quotient of $F(S_{\text{QM}})$ by the relations that hold in $G$ — the congruence $\sim_G$ where $t_1 \sim_G t_2$ iff $t_1$ and $t_2$ evaluate to the same element of $G$.

**Step 3.** We must verify that $\bar{\phi}$ respects this congruence: if $t_1 \sim_G t_2$ in $F(S_{\text{QM}})$, then $\bar{\phi}(t_1) = \bar{\phi}(t_2)$ in $G_{\text{cog}}$.

The relations in $G$ are geometric — two Σ_UL-terms evaluate to the same geometric construction. These relations fall into two categories:

**(a) Universal Σ_UL relations** (hold in every Σ_UL-algebra): negate(negate(a)) = a; compose is associative where defined; invert(invert(r)) = r. These hold in $G_{\text{cog}}$ automatically since it is a Σ_UL-algebra. ✓

**(b) $G$-specific relations** (hold because of the particular geometry of $G$): e.g., two distinct Σ_UL-terms might produce the same geometric construction by coincidence. If $t_1 \sim_G t_2$ via a $G$-specific relation, we need $\bar{\phi}(t_1) = \bar{\phi}(t_2)$ in $G_{\text{cog}}$.

For category (b): $G_{\text{QM}}$ is generated by distinct geometric objects (points at distinct positions, lines at distinct angles, etc. — as verified in §1.4). The $G$-specific identifications among generator-level terms arise from the geometric structure of $\mathbb{R}^2$. The corresponding cognitive generators in $G_{\text{cog}}$ are also realized as distinct geometric objects (by the same argument: distinct positions in glyph space). Thus if two QM terms coincide geometrically, their cognitive images may or may not coincide — but we can **choose** the positions of the cognitive generators to ensure that the same geometric relations hold.

Specifically: the generators of $G_{\text{cog}}$ are realized in the same ambient space $G$ (the full geometric algebra). We choose the specific geometric realizations of the cognitive generators so that the interrelations mirror those of the QM generators. This is possible because $G$ is rich enough (infinite positions, angles, scales available — see the embedding theorem proof in `formal-foundations.md` §3.3, Step 2).

With this choice, $\bar{\phi}$ factors through the quotient $G_{\text{QM}} = F(S_{\text{QM}})/{\sim_G}$ and defines a homomorphism $\phi_{\text{primer}}: G_{\text{QM}} \to G_{\text{cog}}$. $\square$

**Remark on the proof.** The key move is in Step 3(b): we use the freedom in choosing the geometric realization of cognitive generators to ensure that $\phi_{\text{primer}}$ is well-defined. This is not circular — the generators are specified semantically in §1.3, and the geometric embedding theorem guarantees a realization exists. What the proof shows is that among the many possible realizations, at least one makes $\phi_{\text{primer}}$ a homomorphism. The choice of realization is the "gauge freedom" in the metaphor — different geometric renderings of the cognitive domain correspond to different but equivalent formulations of the same metaphor.

#### 2.4.5 Erlangen Level of $\phi_{\text{primer}}$

**Claim [CONJECTURED].** $\phi_{\text{primer}}$ preserves structure at **Level 3 (Affine)**.

**Evidence:**

The key idea: the metaphor morphism $\phi_{\text{primer}}: G_{\text{QM}} \to G_{\text{cog}}$ defines how meanings change when moving from the QM domain to the cognitive domain. The connection's domain component is the **infinitesimal generator** of this finite transformation.

### 3.2 Extended Toy Model: Two-Dimensional Context Space

We extend the toy model from `foundation-securing.md` §2 to two dimensions.

**Context space.** $X = [0,1] \times [0,1]$, with coordinates:
- $x_1$ = domain axis (from finance at 0 to geography at 1) — the original axis from Sprint 3
- $x_2$ = abstraction axis (from QM-literal at 0 to cognitive-metaphorical at 1) — the new axis

**Vocabulary.** Four atoms (extending the Sprint 3 vocabulary):

- $a_2$ = RIVER (entity, original)

- Axis 1: financial salience
- Axis 2: geographical salience
- Axis 3: abstract/cognitive salience

### 3.3 Meaning Assignments in the Extended Model

**BANK** (varies along $x_1$, invariant along $x_2$):

$$\llbracket \text{BANK} \rrbracket_{(x_1, x_2)} = \begin{pmatrix} 1 - x_1 \\ x_1 \\ 0 \end{pmatrix}$$

**RIVER** (constant):

$$\llbracket \text{RIVER} \rrbracket_{(x_1, x_2)} = \begin{pmatrix} 0 \\ 1 \\ 0 \end{pmatrix}$$

**MONEY** (constant):

$$\llbracket \text{MONEY} \rrbracket_{(x_1, x_2)} = \begin{pmatrix} 1 \\ 0 \\ 0 \end{pmatrix}$$

**STATE** (varies along $x_2$, invariant along $x_1$):

$$\llbracket \text{STATE} \rrbracket_{(x_1, x_2)} = \begin{pmatrix} 0 \\ 0 \\ 1 \end{pmatrix}(1 - x_2) + \begin{pmatrix} 0 \\ 0 \\ 1 \end{pmatrix} x_2 \cdot R(\beta x_2) = R(\beta x_2) \begin{pmatrix} 0 \\ 0 \\ 1 \end{pmatrix}$$

where $R(\theta)$ is rotation by angle $\theta$ in the $(x_1, x_3)$-plane:

$$R(\theta) = \begin{pmatrix} \cos\theta & 0 & \sin\theta \\ 0 & 1 & 0 \\ -\sin\theta & 0 & \cos\theta \end{pmatrix}$$

At $x_2 = 0$ (QM-literal): $\llbracket \text{STATE} \rrbracket = (0, 0, 1)^T$ — pure abstract/quantum state.
At $x_2 = 1$: $\llbracket \text{STATE} \rrbracket = (\sin\beta, 0, \cos\beta)^T$ — partially rotated toward financial salience, representing the cognitive metaphor "state of an account" bleeding in.

The parameter $\beta > 0$ controls how strongly the metaphor shifts "state" from abstract to concrete meaning.

### 3.4 The Connection on the Extended Model

The connection is a pair of $3 \times 3$ matrix-valued components:

$$A = A_1(x_1, x_2) \, dx_1 + A_2(x_1, x_2) \, dx_2$$

**Component $A_1$** (along domain axis — inherited from Sprint 3):

$$A_1 = \begin{pmatrix} 0 & -\alpha & 0 \\ \alpha & 0 & 0 \\ 0 & 0 & 0 \end{pmatrix}$$

This rotates in the financial-geographical plane at rate $\alpha$, unchanged from the Sprint 3 model. The third axis (abstract/cognitive) is unaffected by the domain axis connection.

**Component $A_2$** (along abstraction/metaphor axis — NEW):

$$A_2 = \begin{pmatrix} 0 & 0 & -\beta \\ 0 & 0 & 0 \\ \beta & 0 & 0 \end{pmatrix}$$

This rotates in the financial-abstract plane at rate $\beta$. As context moves from QM-literal ($x_2 = 0$) toward cognitive-metaphorical ($x_2 = 1$), meanings rotate from the abstract axis toward the concrete (financial) axis.

**Interpretation of $A_2$:** This is the connection component generated by the test artifact morphism. When the test artifact shifts context from quantum mechanics to cognition, it rotates abstract quantum meanings toward concrete cognitive salience. The rate $\beta$ measures the **metaphor strength** — how strongly the test artifact couples the abstract and concrete axes.

### 3.5 Derivation: $A_2$ from $\phi_{\text{primer}}$

**Theorem 15 (Connection from Metaphor). [PROVEN]** Given a one-parameter family of Σ_UL-homomorphisms $\phi_\lambda: G_{\text{QM}} \to G$ with $\phi_0 = \iota$ (inclusion) and $\phi_1 = \phi_{\text{primer}}$, the connection component along the metaphor axis is:

$$A_2 = \left.\frac{d}{d\lambda}\right|_{\lambda=0} \phi_\lambda \in \mathfrak{g}$$

and this is a well-defined element of the gauge Lie algebra $\mathfrak{g} = \text{Lie}(\text{Aut}(G))$.

**Proof.**

**Step 1: Construct the one-parameter family.** On the entity sort, $\phi_\lambda$ acts on the fiber $\mathbb{R}^3$ as:

$$\phi_\lambda = R(\beta \lambda) = \begin{pmatrix} \cos(\beta\lambda) & 0 & \sin(\beta\lambda) \\ 0 & 1 & 0 \\ -\sin(\beta\lambda) & 0 & \cos(\beta\lambda) \end{pmatrix}$$

At $\lambda = 0$: $\phi_0 = I$ (identity — stay in QM domain).
At $\lambda = 1$: $\phi_1 = R(\beta)$ (rotate from abstract to concrete — the full metaphor).

Each $\phi_\lambda$ is an orthogonal transformation, hence a Σ_UL-automorphism of $G$ (orthogonal transformations preserve the geometric operations, which are defined in terms of incidence, distance, and angle — all preserved by orthogonal maps). Therefore $\phi_\lambda \in \text{Aut}(G)$ for all $\lambda$.

**Step 2: Compute the infinitesimal generator.**

$$A_2 = \left.\frac{d}{d\lambda}\right|_{\lambda=0} R(\beta\lambda) = \beta \left.\frac{d}{d\theta}\right|_{\theta=0} R(\theta) = \beta \begin{pmatrix} 0 & 0 & 1 \\ 0 & 0 & 0 \\ -1 & 0 & 0 \end{pmatrix}$$

This is $\beta J_{13}$, where $J_{13}$ is the generator of rotations in the $(x_1, x_3)$-plane. It is an element of $\mathfrak{so}(3) \subseteq \mathfrak{g}$, hence a well-defined Lie algebra element. ✓

**Step 3: Convention alignment.** The connection 1-form in our gauge theory conventions (from `gauge-bundle-of-meaning.md` §3.3, with coupling absorbed) is $A_\mu \psi = -\frac{d}{d\lambda}\phi_\lambda|_{\lambda=0} \cdot \psi$. The sign convention gives:

$$A_2 = -\beta J_{13} = \begin{pmatrix} 0 & 0 & -\beta \\ 0 & 0 & 0 \\ \beta & 0 & 0 \end{pmatrix}$$

matching the expression in §3.4. $\square$

### 3.6 Curvature Computation

The curvature 2-form on the extended toy model:

$$F_{12} = \partial_1 A_2 - \partial_2 A_1 + [A_1, A_2]$$

Since $A_1$ and $A_2$ are both constant (independent of $x_1, x_2$), the partial derivative terms vanish:

$$\partial_1 A_2 = 0, \qquad \partial_2 A_1 = 0$$

The commutator:

$$[A_1, A_2] = A_1 A_2 - A_2 A_1$$

$$A_1 A_2 = \begin{pmatrix} 0 & -\alpha & 0 \\ \alpha & 0 & 0 \\ 0 & 0 & 0 \end{pmatrix} \begin{pmatrix} 0 & 0 & -\beta \\ 0 & 0 & 0 \\ \beta & 0 & 0 \end{pmatrix} = \begin{pmatrix} 0 & 0 & 0 \\ 0 & 0 & -\alpha\beta \\ 0 & 0 & 0 \end{pmatrix}$$

$$A_2 A_1 = \begin{pmatrix} 0 & 0 & -\beta \\ 0 & 0 & 0 \\ \beta & 0 & 0 \end{pmatrix} \begin{pmatrix} 0 & -\alpha & 0 \\ \alpha & 0 & 0 \\ 0 & 0 & 0 \end{pmatrix} = \begin{pmatrix} 0 & 0 & 0 \\ 0 & 0 & 0 \\ 0 & -\alpha\beta & 0 \end{pmatrix}$$

$$F_{12} = [A_1, A_2] = \begin{pmatrix} 0 & 0 & 0 \\ 0 & 0 & -\alpha\beta \\ 0 & \alpha\beta & 0 \end{pmatrix}$$

**The curvature is non-zero** whenever both $\alpha \neq 0$ and $\beta \neq 0$.

$$F_{12} = \alpha\beta \begin{pmatrix} 0 & 0 & 0 \\ 0 & 0 & -1 \\ 0 & 1 & 0 \end{pmatrix} = \alpha\beta \, J_{23}$$

This is a rotation generator in the **geographical-abstract plane** — a direction that neither $A_1$ (financial-geographical) nor $A_2$ (financial-abstract) individually activate.

### 3.7 Interpretation of the Curvature

**The curvature $F_{12}$ generates rotation in the geographical-abstract plane.** This means:

1. **Parallel transport is path-dependent.** Transporting a meaning along the domain axis (QM→geography) and then along the metaphor axis gives a different result than transporting along the metaphor axis first and then the domain axis. The difference is a rotation by $\alpha\beta \cdot (\text{area enclosed})$ in the geographical-abstract plane.

2. **New meaning direction emerges.** The curvature generates motion in a direction ($J_{23}$: geographical-abstract) that neither connection component alone produces. This is the geometric content of **emergent meaning**: when you metaphorically project quantum concepts AND shift between content domains, a new semantic axis (geographical-abstract blending) is activated that neither process activates independently.

3. **Curvature is proportional to $\alpha\beta$.** The "emergent meaning" is proportional to the product of the domain-shift rate ($\alpha$) and the metaphor strength ($\beta$). If either is zero — no domain crossing or no metaphorical projection — there is no emergent meaning. This matches the test artifact's mechanism: the effect requires BOTH cross-domain activation AND metaphorical structure.

### 3.8 Holonomy on the Extended Model

On $X = [0,1]^2$, consider the rectangular loop $\gamma$ traversing:

$$(0,0) \xrightarrow{x_1} (1,0) \xrightarrow{x_2} (1,1) \xrightarrow{-x_1} (0,1) \xrightarrow{-x_2} (0,0)$$

For constant connection components, the holonomy around a rectangular loop enclosing area $\Delta x_1 \cdot \Delta x_2$ is (to first order):

$$\text{Hol}(\gamma) \approx I + F_{12} \cdot \Delta x_1 \cdot \Delta x_2 = I + \alpha\beta \, J_{23}$$

For the unit square ($\Delta x_1 = \Delta x_2 = 1$):

$$\text{Hol}(\gamma) = \exp(\alpha\beta \, J_{23}) = \begin{pmatrix} 1 & 0 & 0 \\ 0 & \cos(\alpha\beta) & -\sin(\alpha\beta) \\ 0 & \sin(\alpha\beta) & \cos(\alpha\beta) \end{pmatrix}$$

**This is non-trivial whenever $\alpha\beta \notin 2\pi\mathbb{Z}$.** The holonomy is a rotation in the geographical-abstract plane, confirming the Polysemy-Holonomy theorem on this 2D model.

**Physical meaning.** Take the word STATE with meaning $(0, 0, 1)^T$ at the origin (pure abstract meaning in QM-financial context). Transport it around the loop:
- Along $x_1$: it stays at $(0, 0, 1)^T$ (the $A_1$ component doesn't affect the abstract axis)
- Along $x_2$: it rotates toward financial ($\sin\beta, 0, \cos\beta)^T$) 
- Back along $-x_1$: the $A_1$ rotation now affects it (it has financial component)
- Back along $-x_2$: it doesn't return to $(0, 0, 1)^T$

The returned meaning has acquired a geographical component (axis 2) that it didn't start with. This is the holonomy: the word "state," after being metaphorically projected (QM→cognition) and then domain-shifted (finance→geography→finance), acquires a geographical sense ("state" as geographical territory) that was not present in the original QM meaning.

**This is precisely the mechanism the test artifact exploits:** by forcing the reader through both a domain shift and a metaphorical projection simultaneously, new meaning directions emerge that neither shift alone would produce.

### 3.9 Curvature Localization (Qualitative Prediction)

The constant-curvature model of §3.6 is a simplification. In a realistic model, the connection components would vary across $X$:

$$A_1(x_1, x_2), \quad A_2(x_1, x_2)$$

The curvature would then include the derivative terms:

$$F_{12}(x) = \partial_1 A_2 - \partial_2 A_1 + [A_1, A_2]$$

**Qualitative prediction [CONJECTURED]:** The curvature $|F_{12}(x)|$ is concentrated at **domain boundaries** — regions of $X$ where the context transitions between domains (e.g., the point $x_1 = 0.5$ where financial and geographical saliences are equal, or $x_2 = 0.5$ where QM and cognitive interpretations compete).

**Reasoning:** At domain boundaries, both connection components change rapidly: the "frame rotation" that tracks a word's meaning accelerates when the context is ambiguous. Within a single domain, the connection is approximately flat (meanings don't shift much within pure QM or pure finance). The curvature, as the non-abelian part of the field strength, peaks where both components are simultaneously large and non-commuting.

This matches the linguistic observation that polysemy is concentrated at sense boundaries — the point where "bank" transitions from riverbank to financial institution is where the meaning curvature is highest.
