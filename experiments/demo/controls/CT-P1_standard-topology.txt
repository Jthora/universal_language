Let X be a topological space and C_n(X) the free abelian group generated by singular n-simplices σ: Δⁿ → X. The boundary operator ∂_n: C_n(X) → C_{n−1}(X) is defined by ∂_n(σ) = ∑_{i=0}^{n} (−1)^i σ|[v₀,...,v̂ᵢ,...,vₙ], satisfying ∂_{n-1} ∘ ∂_n = 0. The n-th singular homology group is H_n(X;ℤ) = ker(∂_n) / im(∂_{n+1}).

For the n-sphere Sⁿ, H_k(Sⁿ;ℤ) = ℤ if k = 0 or k = n, and H_k(Sⁿ;ℤ) = 0 otherwise. The Mayer-Vietoris sequence for X = U ∪ V gives the long exact sequence: ⋯ → H_n(U∩V) → H_n(U) ⊕ H_n(V) → H_n(X) → H_{n−1}(U∩V) → ⋯

The cohomology ring H*(X;R) with cup product ∪: H^p(X;R) ⊗ H^q(X;R) → H^{p+q}(X;R) satisfies the graded commutativity rule α∪β = (−1)^{pq} β∪α. For the torus T² = S¹×S¹, H*(T²;ℤ) ≅ ℤ[α,β]/(α²,β²) with |α|=|β|=1.

The Euler characteristic χ(X) = ∑_n (−1)^n rank H_n(X;ℤ) equals ∑_n (−1)^n c_n for any CW structure with c_n cells of dimension n. For compact orientable surfaces of genus g, χ = 2 − 2g. The Lefschetz fixed-point theorem gives L(f) = ∑_n (−1)^n tr(f_*: H_n → H_n); if L(f) ≠ 0, then f has a fixed point.
