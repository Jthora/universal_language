# Experiment Workspace

This folder contains prompts for an experiment. Open individual prompt files and paste their contents into a fresh Copilot chat session to run each trial. Save responses in the `results/` folder.

## Instructions

1. Select your model in Copilot (e.g., Claude Opus 4.6)
2. For each prompt file in `prompts/`:
   - Open a NEW Copilot chat session
   - Copy the entire contents of the prompt file
   - Paste into the chat and send
   - Copy the full response
   - Save as a file in `results/` with the same name as the prompt file
3. Do NOT add any additional text, greetings, or follow-up questions
