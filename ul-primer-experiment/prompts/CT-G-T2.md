Let K be an algebraic number field with ring of integers O_K and discriminant Δ_K. The Dedekind zeta function is ζ_K(s) = ∑_{𝔞⊂O_K} N(𝔞)^{-s} = ∏_𝔭 (1 − N(𝔭)^{-s})^{-1} for Re(s) > 1. The class number formula gives lim_{s→1} (s−1)ζ_K(s) = (2^{r₁}(2π)^{r₂}h_K R_K) / (w_K √|Δ_K|), where r₁ and r₂ are the numbers of real and complex embeddings, h_K is the class number, R_K the regulator, and w_K the number of roots of unity.

For a quadratic extension K = ℚ(√d), the ring O_K = ℤ[ω] where ω = (1+√d)/2 if d ≡ 1 mod 4, and ω = √d otherwise. The discriminant is Δ_K = d if d ≡ 1 mod 4, and Δ_K = 4d otherwise.

The ideal class group Cl(O_K) has order h_K. By Minkowski's bound, every ideal class contains an integral ideal of norm ≤ M_K = (n!/nⁿ)(4/π)^{r₂}√|Δ_K|. For K = ℚ(√−23), we have Δ_K = −23, M_K ≈ 3.04, so we check primes p ≤ 3.

The decomposition of rational primes in O_K follows: if p | Δ_K then (p) = 𝔭² ramifies; if (Δ_K/p) = 1 then (p) = 𝔭𝔭̄ splits; if (Δ_K/p) = −1 then (p) remains inert. The Frobenius element σ_𝔭 ∈ Gal(K/ℚ) satisfies σ_𝔭(x) ≡ x^p mod 𝔭 for all x ∈ O_K.
